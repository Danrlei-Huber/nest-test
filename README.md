
## Description

this is an project exemple for run projects nestjs in workspace serverless

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run build

# watch mode -- OFF
$ npm run start:dev

# production mode -- OFF
$ npm run start:prod
```
